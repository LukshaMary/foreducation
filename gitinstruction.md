# Инструкция по GIT

Данная инструкция рассчитана на человека, прежде не работавшего с системами контроля версий, однако, возможно, имеющего представление о том, что такое командная строка.
Здесь будут описаны только самые базовые операции, которым и необходимо владеть для успешной работы над научным проектом на Летней школе. 
Если у вас будут возникать какие­либо вопросы — смело обращайтесь к кураторам проектов за разъяснениями и советами. 

## Базовые команды

*git init

*git commit

*git status
